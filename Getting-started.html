Getting started

1. First step - Firefox

Set yourself up with Firefox (if you don't already have). Any or all of the following versions:
Firefox Beta 49+
Firefox for Android Beta 49+
Firefox for iOS

2. Second step - Add-on

Install the Webcompat.com Reporter add-on
We have a mobile add-on people can install for Fennec Beta and in version 50 there will be a default “Report Site Issue” menu item in Fennec Aurora and Nightly.
No add-ons support for iOS, but people can report bugs manually at webcompat.com

3. Third step - Find sites you want to test

Now you have to find some sites you want to test, we recommend the top sites in your country, as measured by a reputable local source or using the Alexa Top Sites by Country reference. You can also find a list of websites from your location and test them!
In the next page we will discuss how find things that are really a issue.
