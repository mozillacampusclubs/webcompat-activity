Finding issues on websites

By @felipez3r0 - Last updated: Dec/2016

What is a compatibility issue?

First of all we need to understand, what is a compatibility issue?

  1. You visit a webpage.
  2. Something about the experience is broken: layout is messed up, video doesn't play, buttons don't work, etc.
  3. Check in another browser. Is it working there but not in Firefox?

You can help!


Important note: We try to avoid working on fixing Web sites which are working badly everywhere. We focus first our energy on Web sites which are broken in one browser and not others. Issues caused by add-ons or ad blockers are also not considered web compatibility issues.

How to identify a real compatibility issue?

Some problems on websites are not considered a web compatibility issue and this is really important!
Let's discusss some errors that are NOT a compatibility issue:

  1. The website is not responsive in all browsers;
  2. Issues caused by add-ons or ad blockers are also not considered web compatibility issues;
  3. Error/warning messages in the Web console are not a web compatibility issue (but can help to identify one)

I found a web compatibility issue, what I do now?

When you start testing sites and look for issues usually you will find some! The best approach to confirm that you have found a issue is to test in many browsers as possible to check if it's a compatibility problem.
